Linux Overview

The information that follows is neither exhaustive nor authoritative. It is just personal experiences, observations and research gathered by this author in his continual learning process about Linux OS's. Some of this data will be incomplete and perhaps even misleading or wrong; this is entirely unintentional. As stated earlier this is the briefest of Polar 50,000 ft. views of Linux and is only meant as a starting point to pique the reader's curiosity. If any one of these distributions or even Linux as a whole interest you, please further research and try it out. After all, it is free to download and install. Part of the fun of learning and using a Linux OS is breaking it and putting it back together again. Enjoy.

Debian family

Discussion of the Debian family of linux ditributions has to start with the Linux giant Ubuntu from Canonical. Basically, Debian is made up the Ubuntu distros(short for distribution of "flavor" of Linux) along with the Knoppix branch of distros with a myriad of other distros that are Debian based but not directly tied to Ubuntu/Canonical or Knoppix.

The Ubuntu/Canonical family of Linux distros is perhaps the most widely used of the Linux distros; at least for the the newer Linux user, since it has such a Microsoft Windows look and feel. These Ubuntu based distros are built in such a way that terminal, or command line, use is not mandatory. Updates and Upgrades along with downloading and installing new software can all be done in a handy Software window which can be brought up through a icon hyperlink in the toolbar. This toolbar also contains a start button type icon, similar to a Microsoft windows system. If one wants to use command line(terminal in Linux speak), that is possible too and example terminal Linux bash code will follow later.

sudo apt-get update

This author started on Ubuntu himself and was able to slowly learn and explore the Linux world. Other distros that area interesting are Linux Mint and Bodhi Linux, Kubuntu, and Lubuntu. the last three are minimalist Ubuntu based distros that are built for smaller, older machines. Indeed, an old XP machine is still running strong on Lubuntu and an old netbook was revived via Bodhi Linux.

Slackware spinoffs

Not much can this author tell you Slackware as this family of distros has not yet been tried personally. The package managment system for Slackware is pkgtools. A bit of research reveals these distros to strive to Unix standards and feel, and this fact piques this author's curiosity. If one is to do Linux, might as well use a distro version close to Unix, hey? With the twin aims of stability and simplicity, this family of distros is worth a try.

slackpkg update

Worth mentioning here is the another popular Linux branch the SuSE family of distros. Most noteworthy would be Open Suse; as this is used in enterprise/commercial settings. This author has heard positive things about slackware through the years; this is definitely on the bucket list.

zypper update
Red Hat family

What more can be said than RHEL. Yes, it seems with all the countless number of Linux distros out there; it is the One, Red Hat Enterprise Linux which stands above all else like a Colossus in the business and commercial world. It is this distribution that has the most popular and respected certifications to get well paying system administrator jobs.

sudo dnf/yum update

Red Hat distros

  • Red Hat Enterprise Linux
  • CentOS
  • Fedora
  • Mandriva
  • Mageia
  • ROSA

Fedora is the current OS for this author. It is the buggy, cutting edge distro of the Red Hat family. If one were to want the client side or "free" distro of RHEL, then CentOS would be the choice. Fedora has been a fun journey of learning and exploration. Once, this author broke the Xserver back in Fedora 25/26?. After some good research and learning about the Xshell and some of the underlying boot loader processes; this author found a way to restore the corrupted bin files, or rather, the links to start the Xserver and gui processes again. In a odd way, it seems this author has come full circle again in considering to change into CentOS. CentOS was the very first taste of Linux that this author took back in a class at school.

Gentoo and Arch

Gentoo is said to be the Linux for advanced linux users, the sys admins and developers. It's package management system Portage allows for numerous custom optimizations for the user and the ability to turn off unwanted specifications for any software package.

emerge --sync

In a very similar way, Arch and its various distros like Manjaro allow for much customization of the the Linux enviroment for the user. Pacman is the Arch Linux package manager. It is a stable, secure and fast and the ability to customize with the AUR Arch User repository makes it a a popular selection of the Linux community.

pacman -Syu
Android generation and Others

The popular mobile Android mobile phones have a heart of the Linux kernel, but there is plenty of dispute over whether these Android devices house a genuine Linux OS or not. Here is an interesting discussion about this, Is Android Linux you can decide for yourself how to classify these Android systems.

Other distros of note are Chrome OS; Red Hat derivatives Oracle and Scientific Linux; Mandriva derivatives such as Mageia and ROSA; and finally the famous penetration testing platform Kali Linux.
